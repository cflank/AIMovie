# 示例数据

本目录包含AIMovie Cloud的示例数据和测试文件。

## 📁 目录结构

```
samples/
├── videos/          # 示例视频文件
├── outputs/         # 示例输出结果
└── configs/         # 示例配置文件
```

## 🎬 示例视频

- `sample_video.mp4` - 5分钟示例视频
- `short_clip.mp4` - 30秒短视频测试

## 🚀 快速测试

1. 启动服务: `python start.py`
2. 访问Web界面: http://127.0.0.1:8501
3. 上传示例视频进行测试

## 📊 预期结果

- 视频分析时间: 1-3分钟
- 解说生成时间: 10-30秒
- 语音合成时间: 30-60秒
- 总处理时间: 2-5分钟
